## Initialize
> git init

## Add Origin
> git remote add origin 'repo url'

## For existing project
> git clone 'repo url'

## Set Username
> git config --global user.name

## Set Useremail
> git config --global user.email

## Get Status
> git status
    - red colour > file in unstage
    - green colour > file in stage

## To Remove Tracked File
> git rm --cached <file>...

## Add
> git add .

## Commit
> git commit -m "message"
